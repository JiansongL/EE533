# EE533 Labs

This directory contains all laboratory assignments for EE533.

## Structure

Each lab should be placed in its own directory:
- `lab1/` - Lab 1
- `lab2/` - Lab 2
- `lab3/` - Lab 3
- etc.

## Lab Submission Guidelines

1. Create a new directory for each lab
2. Include all source code, data files, and documentation
3. Add a README.md in each lab directory explaining the lab contents
4. Ensure all code is properly commented
5. Include any output files or results as specified in the assignment
